Features

Everything you need, nothing more.

Built for developers who live in the terminal and value privacy.

🔍

Natural language recall

Ask questions in plain English. Local embeddings + LLM reasoning find exactly what you saved, even weeks later.

🔒

Fully local & private

All AI runs via Ollama on your machine. Zero cloud, zero accounts, zero telemetry. Your data never leaves.

⏰

Smart reminders

Parse natural time expressions like "in 30 minutes", "tomorrow 9am", "Friday 3pm" into real deadlines.

🏷️

Auto-enriched metadata

Every memory captures working directory, hostname, and timestamp. AI auto-detects type and extracts tags.

📴

Offline-safe

Memories save even when Ollama is unavailable. AI enrichment gracefully degrades — your data is never lost.

🧩

Fully extensible

Ports & Adapters architecture. Swap storage, AI backend, or notifier by implementing a single interface.

Installation

Up and running in minutes.

Requires Go 1.21+ and Ollama running locally.

1

Install Ollama and pull models

Get Ollama from ollama.com, then pull the required models.

ollama pull nomic-embed-text   # embeddings
ollama pull llama3.2:3b        # reasoning (swap for any chat model)
2

Install yaad

go install github.com/kunalsin9h/yaad/cmd/yaad@latest
3

Initialise config

yaad config init
4

Save your first memory

yaad add "hello world" --for "my first yaad memory"
Usage

A command for every moment.

Five memory types, auto-detected by AI: command · note · url · fact · reminder

yaad add "<content>" [flags]

Save a memory. Use --for for context, --remind for a deadline, --tag for labels.

yaad ask "<question>"

Query everything you've saved with natural language. The LLM reasons over semantic search results.

yaad list [flags]

Browse recent memories. Filter by --type, --tag, or --remind (pending reminders only).

yaad get <id>

Show full details of a memory. Accepts 10-char ULID prefix — no need to type the full ID.

yaad delete <id>

Delete a memory by ID. Prompts for confirmation; skip with -y.

yaad config init / set / get

Manage ~/.yaadrc. Override Ollama URL, models, notifier, and reminder poll interval.

# a command to resume later
yaad add "claude --resume 17a43487" --for "yaad build session"

# a fact with tags
yaad add "staging postgres is on port 5433" --tag postgres --tag staging

# a time-sensitive reminder
yaad add "submit PR for review" --remind "tomorrow 9am"

# recall anything
yaad ask "what was the staging postgres port?"
yaad ask "do I have anything due tonight?"
Reminders

Never forget a deadline.

Two ways to surface reminders — pick what fits your workflow.

Recommended

Inline — shell PROMPT_COMMAND

Reminders surface directly in your terminal on every prompt. No background process required.

# ~/.bashrc
export PROMPT_COMMAND="yaad check; $PROMPT_COMMAND"

# ~/.zshrc
precmd() { yaad check }
Systemd

Background daemon

Install yaad as a systemd user service. Sends desktop notifications via notify-send automatically.

yaad daemon install
systemctl --user enable --now yaad

# check status
systemctl --user status yaad
